# 📋 TaskMaster Pro - Gestão de Tarefas Inteligente

Um organizador de tarefas robusto e elegante, focado em produtividade e
organização pessoal. Este projeto combina uma interface moderna (Dark Mode) com funcionalidades avançadas de gestão de fluxo de trabalho.

## 🚀 Funcionalidades

- ✅ **Gestão de Tarefas**: Criação, edição e conclusão de tarefas com subtarefas.
- ✅ **Categorização Dinâmica**: Separação por categorias como Trabalho, Pessoal, Estudos e Saúde.
- ✅ **Sistema de Prioridades**: Níveis de prioridade visual (Alta, Média, Baixa).
- ✅ **Múltiplas Visualizações**: Alternância entre Lista, Kanban e Calendário.
- ✅ **Gestão de Dados**: Funções para Exportar, Importar e Resetar o banco de tarefas.
- ✅ **Interface Moderna**: Design Dark Mode limpo e responsivo.

## 🛠️ Tecnologias Utilizadas

- **HTML5 / CSS3 / JavaScript**
- **React** (Componentização e Hooks)
- **Tailwind CSS** (Estilização Utilitária)
- **Lucide Icons** (Ícones da Interface)

O código está organizado de forma modular para facilitar a manutenção e escalabilidade, seguindo as melhores práticas de desenvolvimento Front-End.
